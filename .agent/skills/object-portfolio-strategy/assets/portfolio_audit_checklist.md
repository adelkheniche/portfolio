# Portfolio audit checklist

## Structure
- [ ] Le positionnement est clair en un paragraphe.
- [ ] Le portfolio principal contient 3 à 5 projets forts.
- [ ] L’ordre des sections est logique.
- [ ] Une distinction existe entre couche principale et couche de preuve.

## Proof
- [ ] Chaque projet montre objet + process + compétence + fabrication + évaluation.
- [ ] Le rôle est explicite.
- [ ] Le statut du projet est explicite.
- [ ] Il y a une fabrication proof.
- [ ] Il y a une repeatability statement.

## Writing
- [ ] Le ton est précis et calme.
- [ ] Les adjectifs vagues ont été remplacés par des preuves.
- [ ] Les textes sont courts et lisibles.
- [ ] Les paragraphes ne sonnent ni corporate ni flous.

## Images
- [ ] Il y a un hero shot.
- [ ] Il y a une image d’usage ou d’échelle.
- [ ] Il y a des detail shots.
- [ ] Les images de process montrent de vraies décisions.
- [ ] Les fonds et la lumière sont cohérents.

## Credibility
- [ ] Il n’y a pas d’ambiguïté sur l’auteur.
- [ ] Les crédits sont mentionnés quand nécessaire.
- [ ] Les compétences revendiquées sont reliées à des projets.
- [ ] Le portfolio montre ce qui peut être refait professionnellement.
