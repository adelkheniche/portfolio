# REFERENCE — Conventions d’écriture

## Principes
- Procédures > déclarations.
- Preuves > adjectifs.
- Projets profonds > accumulation de projets superficiels.
- Portfolio principal édité > documentation brute.
- Démarche personnelle traduite en décisions concrètes.

## Structure recommandée
1. Cadrage
2. Sélection
3. Structure
4. Écriture
5. Validation

## Rappels métier
Un portfolio d’objets doit prouver en même temps :
- l’objet,
- le process,
- la compétence,
- la fabrication,
- l’évaluation.

## Positionnement conseillé
Quand l’utilisateur veut une voix personnelle forte et une lisibilité professionnelle :
- authorial product maker.
